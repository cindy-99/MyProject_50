# 🌸 花卉信息管理系统

一个基于Java开发的简单花卉信息管理系统，可以帮助用户管理花卉信息，包括添加、查看、搜索和删除花卉记录。

## 📖 项目概述

本项目是一个控制台应用程序，使用Java编写，采用面向对象的设计理念。系统允许用户管理花卉的基本信息，如名称、颜色、开花季节和价格等，为花卉爱好者或零售商提供一个简单的信息管理工具。


## ✨ 功能特性

- **花卉信息添加** - 记录花卉的名称、颜色、开花季节和价格
- **花卉信息浏览** - 以表格形式清晰展示所有花卉信息
- **花卉搜索功能** - 根据名称快速查找特定花卉
- **花卉信息删除** - 移除不需要的花卉记录
- **数据持久化** - 使用集合框架在内存中管理数据

## 🛠️ 技术栈

- **语言**：Java
- **开发环境**：标准Java开发环境 (JDK 8+)
- **数据结构**：ArrayList集合
- **构建工具**：纯Java，无需额外依赖
